# GitHub Pages publishing directory

This `docs/` directory mirrors the minimal SonaKeep static site so GitHub Pages can publish directly from:

- Branch: `main`
- Folder: `/docs`

## Files
- `index.html`
- `privacy.html`
- `support.html`

## GitHub Pages settings
After this branch is merged:

1. Open **Settings → Pages**
2. Set **Source** to **Deploy from a branch**
3. Select **main**
4. Select **/docs**
5. Click **Save**

Your Pages URLs should then resolve under:

- `/sonakeep-649/`
- `/sonakeep-649/privacy.html`
- `/sonakeep-649/support.html`
