# SonaKeep Release Packet

This directory contains the final repository-side release artifacts for the last mile before App Review.

## Files
- `final-go-no-go-checklist.md` — release gate and decision checklist
- `app-store-submission-runbook.md` — manual submission runbook for App Store Connect

## What still needs to happen manually
- Run the final GO / NO-GO review against the actual release-candidate build
- Confirm live URLs, screenshots, and metadata in App Store Connect
- Submit the selected build for review
- Respond to any App Review questions or rejection notes

## Recommended order
1. Complete the TestFlight cycle
2. Run the final go / no-go checklist
3. Use the App Store submission runbook
4. Submit the build
